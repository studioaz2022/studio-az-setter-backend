## Domain Advantage Reminder (3 Kings Rule)

The domain `tattooshopminneapolis.com` literally IS the #1 keyword "tattoo shop Minneapolis." This means:

- URL slugs must NOT repeat "tattoo" — the domain already provides that signal
- Every page URL automatically inherits keyword relevance from the domain
- Title tags can use the saved characters for secondary keywords or brand name
- This is our single biggest SEO advantage over every competitor in the market
